phenotype module¶

class phenotype.Phenotype(phenotype)¶
add_genome(genome)¶

Takes in a genome object and if it has the same phenotype as the phenotype instance allows that genotype to be added to the phenotypes list of genomes (self.genomes)

compare_conserved_sequences(other_phenotype)¶
get_conserved_sequences()¶

Using some metrics and methods pulls out the conserved non-coding sequences from a collection of genomes stored in this phenotype instance.
